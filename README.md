Next.js Project

Note: The Project is Uploaded without node modules folder

This project is built using Next.js.

Getting Started

Before running the project, make sure to install the required dependencies:

npm install axios
npm install react-icons 

Then, start the development server:

npm run dev

Project Routes & Features


Task1:

When you open     http://localhost:3000   you will find task1.

Task1 is focused on converting a Figma design into a fully responsive Next.js UI.



Task2:

Register Page

Navigate to:

http://localhost:3000/register 

This will open the Register page.

Users must enter valid data in order to complete the registration process successfully.

Verify Page

After successful registration, the user will be redirected to:

http://localhost:3000/verify 

This page is used to verify the user.

Dashboard

Once verification is completed, the user will be redirected to:

http://localhost:3000/dashboard 

The Dashboard page displays a welcome message:

Hello + User Name
